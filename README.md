# Init My RPi4
A collection of tools to make RPi4 more powerful. 

## Run
```
$ git clone https://github.com/veerendra2/init-my-rpi4.git
$ cd init-my-rpi4
## Update inventory file
$ ansible-playbook -i inventory.yml init.yml
```
